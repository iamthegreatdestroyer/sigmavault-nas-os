================================================================================
IF SOMETHING BREAKS: TROUBLESHOOTING
================================================================================

ERROR: "ModuleNotFoundError: No module named 'gi'"
FIX: Install PyGObject system package
     On Ubuntu/Debian: sudo apt install python3-gi python3-gi-cairo gir1.2-gtk-4.0 gir1.2-adwaita-1
     On Fedora: sudo dnf install python3-gobject gtk4-devel adwaita-devel

ERROR: "Error creating window: couldn't find required GTK4 libraries"
FIX: Verify GTK4 installed
     python -c "import gi; gi.require_version('Gtk', '4.0'); gi.require_version('Adw', '1'); print('‚úÖ Ready')"

ERROR: "Could not connect to API - Connection refused"
FIX: Make sure Go API is running in separate terminal
     curl http://localhost:12080/api/v1/health
     (Should see: {"status":"healthy",...})

ERROR: "Port 12080 already in use"
FIX: Kill existing process
     On Windows: netstat -ano | findstr 12080
                 taskkill /PID [PID] /F
     On Linux/Mac: lsof -i :12080
                   kill -9 [PID]

ERROR: "ImportError or blank window"
FIX: Delete venv and rebuild
     rm -rf .venv
     python -m venv .venv
     source .venv/bin/activate  (or .venv\Scripts\activate on Windows)
     pip install --upgrade pip setuptools
     pip install -e .
