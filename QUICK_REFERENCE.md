## 🐛 TROUBLESHOOTING

```powershell
# Port already in use
netstat -ano | Select-String ":5000"   # Find PID
netstat -ano | Select-String ":12080"  # Find PID
Stop-Process -Id <PID> -Force          # Kill process

# Services not starting
.\scripts\dev-environment-setup.ps1 -Kill    # Clean slate
.\scripts\dev-environment-setup.ps1 -Full    # Rebuild

# Desktop UI issues
.\scripts\launch-desktop-msys2.ps1 -Check    # Verify packages
.\scripts\launch-desktop-msys2.ps1 -Install  # Fix packages
```

## 💡 PRO TIPS

- Use `-Quick` for daily startup (assumes deps installed)
- Keep a terminal with `-Kill` command ready
- Desktop UI restart: `Ctrl+C` → `Up Arrow` → `Enter`
- Check logs in background with `Get-Job | Receive-Job`
- Use GTK Inspector: Set `$env:GTK_DEBUG='interactive'`
